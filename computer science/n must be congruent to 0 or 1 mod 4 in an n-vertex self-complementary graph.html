n must be congruent to 0 or 1 mod 4 in an n-vertex self-complementary graph

complement graph : the complement or inverse of a graph G is a graph H on the same vertices such that two distinct vertices of H are adjacent if and only if they are not adjacent in G, it means that one fills in all the missing edges required to form a complete graph, and removes all the edges that were previously there.

Self-complementary graph : a self-complementary graph is a graph which is isomorphic to its complement.

Last night, one of my schoolmates asked a question that is a proof about the n-vertex self-complementary graph's n must meet that n must be congruent to 0 or 1 mod 4, in other words, an n-vertex graph (if there is more than one vertex) is a self-complementary graph if and only if

n = 4k or n = 4k + 1 (k ∈ Z+)

acording to the principle of self-complementary graph, we can know that an n-vertex self-complementary graph has exactly half number of edges of the complete graph.

we assume there is an n-vertex self-complementary graph G, H is G's complement graph, mG is G's number of edge, mH is H's number of edge, we can get that

mG = mH (m ≥ 0, m ∈ Z)

mG + mH = n * (n - 1) / 2

4m = n * (n - 1)

m = (n / 4) * (n - 1)

or can write as

m = ((n - 1) / 4) * n

we get

n/4 ∈ Z+

or

(n - 1)/4 ∈ Z+

set k ∈ Z+, we can conclude it must be n = 4k or n = 4k + 1
