3.more efficiently

To make you work more efficient, you neeed learmed several basic commands.

3.0 switch to root
su root

3.1 set or change the root password
su do passwd
input the password of the user which was set firstly when you install this system
input the (new) password for the root

3.2 install software(root)
apt-get install xxx

3.3 instal software(genural user)
sudo apt-get xxx

3.4 install from deb pakage
dpkg -i xxx.deb

3.5 extract tar.gz files
tar -zxfv xxx.tar.gz

3.6 show dependence of deb pakage
gdebi xxx.deb

3.8 copy file to another directory
cp /a/b /c/
(note : copy the b file of directory a to directory c )

3.8 move a file to a (new) directory
mv /a/b /c/
(mote : refer to last noe)

3.9 delete a file
rm /a/b

3.10 uninnstall software
apt-get remove xxx
uninstall the xxx but don't remove the relative configure
apt-get purge xxx
uninstall the xxx and the relative configure

To lead you work more efficient, you neeed customize you system
